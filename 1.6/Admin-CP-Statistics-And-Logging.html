Tools & Maintenance: Cache Manager

The Cache Manager provides a listing of all of the caches on your board, the size of each cache, and allows you to rebuild caches that have information that could become corrupted.

Clicking on the name of the cache will take you to a page that displays the contents of that cache.

Tasks are PHP scripts run at set intervals which typically maintain your database and board, as well as perform tasks for some features, such as mass mail and group promotions. Read the full document »

Tools & Maintenance: Recount and Rebuild

The Recount and Rebuild portion of the Admin CP allows you to recount the various statistics, counters, and attachment thumbnails. This corrects any synchronization errors that may have occurred. This includes Recount Statistics, Rebuild Forum Counters, Rebuild Thread Counters, Recount User Post Counts, and Rebuild Attachment Thumbnails.

Each (excluding Recount Statistics) allows you to specify the number of data entries to process in each page. If there are fewer than the number entered to process, only one page will be used.

To run, enter the number of data entries you wish to process per page, and select Go in the row for the item you wish to recount or rebuild. If more than one page is necessary, you will need to select "Proceed" after each page is run.

The PHP Info page displays the results of the phpinfo() function. This provides you with all of the information regarding your PHP installation.

You can easily backup your forum database to your computer or to the server. Read the full document »

Tools & Maintenance: Optimize Database

The Optimize Database page allows you to reclaim deleted space in the selected database tables. You can choose to Select All tables, Deselect All tables, Select Forum Tables (only select those with the table prefix specified for your MyBB installation - mybb_ by default), or select each table individually by holding down Control.

The File Verification page allows you to check all files in your forum installation against checksums of the correct files, checking for possible corruption. However, this tool cannot differentiate between customizations and corruption; it simply returns files that are different from the default files.
